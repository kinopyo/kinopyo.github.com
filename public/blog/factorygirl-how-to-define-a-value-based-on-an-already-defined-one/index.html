FactoryGirl: How to Define a Value Based on an Already Defined One

FactoryGirlで既に定義した値を使おうとする時の方法です。 例えばLessonというクラスのstart_atとend_atの属性があり、start_atを定義した後はend_atをその2時間後に定義したい場合、ブロックでパラメータを渡せばそこでstart_atにアクセスできるようになります。
