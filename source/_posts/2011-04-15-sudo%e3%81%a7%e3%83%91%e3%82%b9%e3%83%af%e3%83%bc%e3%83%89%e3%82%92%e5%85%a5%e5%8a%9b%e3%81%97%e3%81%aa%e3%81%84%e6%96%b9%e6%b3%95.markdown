ターミナルで
sudo visudo

そして次のラインを追加。("kinopyo"は自分のユーザ名に変更してください)

```
kinopyo ALL=(ALL) NOPASSWD: ALL

```

# 中文
先在Terminal输入
sudo visudo

接着插入下面的代码。("kinopyo"是你的用户名)

```
kinopyo ALL=(ALL) NOPASSWD: ALL

```

# English

Type this command first.
sudo visudo

Then add this line at the end (change “kinopyo” to your username):

```
kinopyo ALL=(ALL) NOPASSWD: ALL

```
