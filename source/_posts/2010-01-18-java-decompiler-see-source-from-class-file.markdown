Javaの.classから逆コンパイラしてもともとのソースコードが見れるツールです。
ずいぶん前一度使ったことがあります。
スタンドアロンのJD-GUIツールと、
Eclipseのプラグインとして使えるJD-Eclipseがあるらしいです。
スタンドアロンはJRE環境が必要とせずインストールだけで使えます。
jarファイルもサポートされているようです。
